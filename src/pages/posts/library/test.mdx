  ### Description
   El mos algorithm es un algoritmo que permite responder consultas offline en un rango de un arreglo. 
   Este algoritmo suele ser más eficiente que un segment tree o fenwick tree en algunos casos.
        
  ### Complexity

   - <Latex formula='O((n + q) * sqrt(n))' className="text-lime-400"/>
