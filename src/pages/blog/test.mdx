# 🌲 Segment Tree

<div class="prose prose-sm prose-lg prose-blue dark:prose-invert mx-auto max-w-3xl p-6 bg-gray-800 text-white rounded-lg shadow-lg">

El **Segment Tree** es una estructura de datos que permite realizar consultas y actualizaciones en rangos de forma eficiente.

</div>

## 📌 Conceptos Clave

- ✅ Se basa en un árbol binario.
- ✅ Permite actualizar valores en `O(log n)`.
- ✅ Permite realizar consultas en `O(log n)`.

## 📝 Implementación en C++

<div class="prose prose-sm prose-lg prose-blue dark:prose-invert mx-auto max-w-3xl p-6 bg-gray-800 text-white rounded-lg shadow-lg">
```cpp
#include <bits/stdc++.h>
using namespace std;

struct SegmentTree {
    vector<int> tree;
    int n;
    
    SegmentTree(int size) {
        n = size;
        tree.resize(4 * n);
    }

    void build(vector<int>& arr, int node, int start, int end) {
        if (start == end) {
            tree[node] = arr[start];
        } else {
            int mid = (start + end) / 2;
            build(arr, 2 * node, start, mid);
            build(arr, 2 * node + 1, mid + 1, end);
            tree[node] = tree[2 * node] + tree[2 * node + 1];
        }
    }
};
```
</div>
